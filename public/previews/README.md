# Preview Pages Assets

This directory contains the preview page images for the Preview page at `/preview`.

## File Naming Convention

Place your preview page images here using the following naming format:

- `page-01.jpg` - Preview page 1
- `page-02.jpg` - Preview page 2
- `page-03.jpg` - Preview page 3
- `page-04.jpg` - Preview page 4
- `page-05.jpg` - Preview page 5

## Image Requirements

- **Format**: JPG or PNG (JPG recommended for file size)
- **Recommended dimensions**: 800px width × 1066px height (3:4 aspect ratio)
- **File size**: Keep under 500KB per image for optimal loading
- **Quality**: High quality but optimized for web

## Adding Your Preview Pages

1. Export your preview pages as images (JPG or PNG)
2. Name them according to the convention above (`page-01.jpg`, `page-02.jpg`, etc.)
3. Place them in this `/public/previews/` directory
4. The Preview page will automatically load them in order
